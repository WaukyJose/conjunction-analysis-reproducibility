Exploratory predictive analysis: modelling-table decisions

The exploratory predictive analysis used complete-case modelling tables.

COREFL and GiG included only texts with available indices from all three discourse levels:
- intra-sentential
- inter-sentential
- inter-paragraph

EFCAMDAT included only intra-sentential and inter-sentential indices because paragraph boundaries were not treated as reliable for inter-paragraph analysis.

Missing inter-paragraph values were not imputed as zero, because unavailable paragraph structure is conceptually different from zero paragraph-initial conjunction use.

- efcamdat_modeling_table_complete_cases.csv
