# Paper outputs

This folder contains safe output files used to support the analyses reported in the paper.

Included outputs:
- `text_level_indices/`: text-level numeric indices by corpus and discourse level.
- `summary_outputs/`: item-level and macro-category summaries by corpus and discourse level.
- `inferential_statistics/`: statistical test outputs.
- `predictive_models/`: exploratory predictive modelling outputs.

The files included here do not contain original corpus texts, full sentences, paragraphs, learner identifiers, or sensitive metadata.

Case-level files containing sentence or paragraph text from the original corpora are not redistributed because of licensing and privacy restrictions.
