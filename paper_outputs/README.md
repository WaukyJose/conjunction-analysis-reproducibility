# Paper outputs

This folder contains public-safe outputs used to support the manuscript analyses.

Included:
- corrected zero-inclusive text-level index outputs
- descriptive summary tables
- inferential statistics outputs
- exploratory Random Forest predictive-analysis outputs
- validation summaries and documentation

Not included:
- raw corpora
- full learner texts
- sentence-level case files
- paragraph-level case files
- learner identifiers or sensitive metadata

Files containing full sentences, paragraphs, learner identifiers, or restricted corpus data are excluded for licensing and privacy reasons.
