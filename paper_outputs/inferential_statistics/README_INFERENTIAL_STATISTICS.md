# Inferential statistics outputs

This folder contains inferential statistical outputs used in the paper.

Included files:
- `intersentential_kruskal_results.csv`
- `intersentential_spearman_correlations.csv`
- `interparagraph_kruskal_functional_results.csv`
- `interparagraph_spearman_correlations.csv`

These files report Kruskal-Wallis and Spearman correlation analyses for inter-sentential and inter-paragraph conjunction indices.

The files contain aggregated statistical outputs only. They do not contain original corpus texts, full sentences, paragraphs, learner identifiers, or sensitive metadata.
