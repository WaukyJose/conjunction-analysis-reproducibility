# Inferential statistics outputs

This folder contains descriptive and inferential statistical outputs used in the paper.

Included outputs cover:
- corpus overview and verified word counts
- descriptive summaries by corpus and developmental group
- intra-sentential conjunction distribution
- intra-sentential Kruskal-Wallis results
- intra-sentential Spearman developmental trends
- macro-category and taxis percentage summaries
- inter-sentential Kruskal-Wallis and Spearman results
- inter-paragraph Kruskal-Wallis and Spearman results

These files contain aggregated statistical outputs only. They do not contain original corpus texts, full sentences, paragraphs, learner identifiers, or sensitive metadata.

Note:
The intra-sentential files marked `priority_fix` correspond to the corrected parser version in which high-frequency duplicated connectors such as `and`, `but`, and `because` were assigned using priority mappings rather than first-match dictionary order.
