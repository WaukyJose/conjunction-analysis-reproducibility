# Dictionary audit files

This folder documents the Halliday-based conjunction inventories used by ConjuncTool.

Files included:

- `intra_dictionary_inventory.csv`  
  Inventory of conjunctions used for intra-sentential analysis.

- `intersent_dictionary_inventory.csv`  
  Inventory of conjunctions used for inter-sentential analysis.

- `paragraph_dictionary_inventory.csv`  
  Inventory of conjunctions used for inter-paragraph analysis.

- `dictionary_duplicate_connectors.csv`  
  Audit of conjunction forms that appear in more than one functional category or dictionary location.

- `dictionary_index_counts.csv`  
  Summary counts of dictionary entries, macro-categories, subcategories, relations, and taxis values.

- `v2_recursive_dictionary_inventory.csv`  
  Flattened recursive inventory showing the full dictionary path for each connector.

- `v2_recursive_index_counts.csv`  
  Summary of index counts derived from the recursive dictionary structure.

These files are included to make the conjunction classification system transparent and auditable.
