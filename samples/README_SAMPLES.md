# Synthetic sample texts

This folder contains short synthetic texts created for reproducibility testing.

The samples are not taken from COREFL, EFCAMDAT, GiG, or any other corpus. They are included only so that users can test the parser and inspect the expected output format without requiring access to restricted corpus data.

Files:
- `synthetic_intra_text.txt`: short text with sentence-internal conjunctions.
- `synthetic_intersent_text.txt`: short text with sentence-initial conjunctions linking adjacent sentences.
- `synthetic_interpara_text.txt`: short text with paragraph-initial conjunctions.

These files can be used with ConjuncTool to generate example outputs.
