# Sample outputs

This folder contains example CSV outputs generated from the synthetic sample texts in `../samples/`.

These outputs are included to show the expected file structure produced by ConjuncTool without redistributing any real corpus text.

Included output types:
- `*_results.csv`: detected conjunction cases from the synthetic examples.
- `*_item_indices.csv`: item-level index outputs.
- `*_summary_by_item.csv`: summary counts by detected item.
- `*_summary_by_macro.csv`: summary counts by Hallidayan macro-category.
- `*_summary_by_taxis.csv`: taxis summary for supported intra-sentential analysis.
- `*_text_indices.csv`: text-level raw and normalized indices.

These files are for demonstration and reproducibility testing only.
